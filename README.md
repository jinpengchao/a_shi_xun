## 《微家》 - Java与移动智能设备开发


### 项目简介

微家——有深度、有爱的中老年人社区

微家是一款致力于关心中老年人安全、丰富中老年人娱乐生活的移动应用。它能够设置远程闹钟，定时提醒老人按时吃药等；能够语音视频通话，实时交流；也可远程查看父母老人实时、历史轨迹，保障老人的安全；亦有休闲社区，陌生朋友，天南海北，畅所欲言；戏曲老歌、相声评书、美文新闻，应有尽有。
